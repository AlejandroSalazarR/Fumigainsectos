# 🦟 FUMIGAINSECTOS S.A.S. - Landing Page Oficial

Landing Page profesional desarrollada en React para **FUMIGAINSECTOS S.A.S.**, empresa líder en manejo integrado de plagas en Medellín, Antioquia.

## 🚀 Características

- ✅ Single Page Application (SPA) totalmente responsive
- ✅ Diseño moderno y profesional Mobile First
- ✅ Carrusel de imágenes automático en Hero Section
- ✅ Información completa de la empresa y sus servicios
- ✅ 9 servicios especializados de control de plagas
- ✅ Sección de plagas con carrusel interactivo
- ✅ Botón flotante de WhatsApp con menú de opciones
- ✅ Optimizado para SEO y accesibilidad
- ✅ Sin dependencias de base de datos

## 📋 Servicios Ofrecidos

1. **Fumigación** - Aplicación de productos químicos certificados
2. **Desratización** - Control profesional de roedores
3. **Tratamiento de Comején** - Curativos y preventivos
4. **Aspersión o Pulverización** - Insecticidas de alto poder residual
5. **Termonebulización** - Control efectivo de insectos voladores
6. **Nebulización** - Aplicación en ultra bajo volumen

9. **Control de Otros Vertebrados** - Palomas, pájaros y murciélagos
